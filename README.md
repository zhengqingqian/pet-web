#### 前言
比赛项目，简单实现了关于宠物托管、领养的功能。`很多功能尚未完善`，很水。

此仓库为前端代码，后端项目参考 [Gitee 后端代码托管仓库](https://gitee.com/zhaofengling/pet-master/tree/master)


### 项目结构

* components --- 组件包
* config    --- 配置包
    * router.js --- 路由配置
    * store.js  --- 单一状态数
* page  --- 主页面
